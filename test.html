页面滚动锁定演示

本页面展示了如何使用JavaScript锁定用户行为，防止页面下拉或滚动。点击下方按钮体验效果。

技术实现

通过JavaScript动态控制CSS的overflow属性，设置为hidden来禁用页面滚动。

同时，我们使用scrollTo(0, 0)方法将页面重置到顶部，确保用户无法通过其他方式滚动页面。

应用场景

  • 全屏模态框或弹窗
  • 游戏界面
  • 幻灯片演示
  • 需要固定视口的特殊页面
当前状态：页面滚动已解锁

实现代码

// 锁定滚动
function lockScroll() {
    document.body.style.overflow = 'hidden';
    window.scrollTo(0, 0);
    document.getElementById('status').textContent = '当前状态：页面滚动已锁定';
    document.getElementById('status').className = 'status locked';
}

// 解锁滚动
function unlockScroll() {
    document.body.style.overflow = 'auto';
    document.getElementById('status').textContent = '当前状态：页面滚动已解锁';
    document.getElementById('status').className = 'status unlocked';
}

注意事项

锁定页面滚动可能会影响用户体验，请谨慎使用。

在移动设备上，可能需要额外处理触摸事件以防止滚动。

确保提供明显的解锁方式，避免用户被困在锁定状态。
